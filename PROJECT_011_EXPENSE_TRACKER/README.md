# Expense Tracker App 💰

A smart and interactive JavaScript project that helps users manage their finances with ease. This expense tracker allows you to record, categorize, and monitor your spending — making budgeting simple and effective.

## Features

* Add and Remove Transactions: Easily track your income and expenses.
* Dynamic Balance Calculation: Automatically updates your total balance.
* Categorized Entries: Organize transactions by type for better insights.
* Persistent Data Storage: Saves your records locally so data remains after refresh.
* Built with Pure JavaScript, HTML, and CSS: Lightweight, responsive, and dependency-free.
