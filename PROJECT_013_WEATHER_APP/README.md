# Weather App ☀️🌧️

A sleek and intuitive JavaScript project that provides real-time weather information for any city. This app fetches live data and displays temperature, conditions, humidity, and more — all in a clean and user-friendly interface.

## Features

* Real-Time Weather Data: Get up-to-date information using a live weather API.
* Search Any City: Instantly view weather details for locations worldwide.
* Detailed Weather Stats: Displays temperature, weather condition, humidity, wind speed, and more.
* Responsive Interface: Works seamlessly across desktop and mobile devices.
* Built with Pure JavaScript, HTML, and CSS: Fast, lightweight, and smooth.
